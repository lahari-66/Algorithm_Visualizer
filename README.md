# Interactive DSA Algorithm Visualizer

An interactive React application that visualizes data structures and algorithms with animated step-by-step execution. Generate arrays, control speed, and follow along with clear narration and complexity details.

## Features
- Sorting visualizations: Bubble, Merge, Quick, Heap
- Searching visualizations: Linear, Binary (auto-sorts input)
- Dynamic array bars with compare, swap, sorted, and found states
- Speed control, pause/resume, reset, and custom input
- Step explanation panel and complexity display

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
